Brain map#

To open up the brain map I will use Nilearn. It is a package for advanced machine learning techniques to analyze MRI data and I will use it to plot the brain map from the analysis with Neuroquery.

import nilearn

nilearn.plotting.plot_img('face_perception_familiarity_identity_face_face_recognition_prediction_er.nii.gz')

<nilearn.plotting.displays._slicers.OrthoSlicer at 0x7f4370edf518>

Here we can some how see where the predicted locations are for face perception and face recognition.

nilearn.plotting.plot_glass_brain('face_perception_familiarity_identity_face_face_recognition_prediction_er.nii.gz')

<nilearn.plotting.displays._projectors.OrthoProjector at 0x7f43709f8ac8>

This plot shows it a bit more clearly what the main region are.
